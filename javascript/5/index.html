JavaScript DOM (Document Object Model)

JavaScript DOM (Document Object Model) Link to heading

DOM (Document Object Model) là một khái niệm quan trọng trong phát triển web. Nó cung cấp một cách để JavaScript có thể truy cập và thay đổi nội dung, cấu trúc và kiểu dáng của các tài liệu HTML hoặc XML. Khi bạn làm việc với các trang web, bạn sẽ thường xuyên cần sử dụng DOM để thay đổi các phần tử trên trang mà không cần phải tải lại toàn bộ trang.

1. DOM là Gì? Link to heading

DOM là một biểu diễn cấu trúc cây của tài liệu HTML hoặc XML. Mỗi phần tử trong tài liệu được xem như là một nút trong cây DOM. DOM giúp JavaScript truy cập và thay đổi các phần tử HTML, thuộc tính và nội dung của chúng một cách động.

Cấu Trúc DOM Link to heading

Cấu trúc của DOM giống như một cây, trong đó mỗi phần tử trong HTML trở thành một nút (node). Ví dụ, khi bạn có một tài liệu HTML như sau:

<!DOCTYPE html>
<html>
<head>
  <title>Trang web của tôi</title>
</head>
<body>
  <h1>Chào mừng đến với trang web của tôi!</h1>
  <p>Đây là một đoạn văn bản.</p>
</body>
</html>

Cấu trúc DOM của nó sẽ như sau:

Sao chép mã
Document
 ├── html
 │    ├── head
 │    │    └── title
 │    └── body
 │         ├── h1
 │         └── p

Mỗi thẻ HTML (như , , ) và nội dung bên trong chúng trở thành các phần tử (elements) trong cây DOM.

2. Truy Cập và Thao Tác Với DOM Link to heading

JavaScript cung cấp nhiều phương thức để truy cập và thay đổi các phần tử DOM. Dưới đây là một số cách phổ biến để làm việc với DOM.

2.1. Lấy Phần Tử DOM Link to heading

Để truy cập phần tử trong DOM, bạn có thể sử dụng một số phương thức như getElementById(), getElementsByClassName(), getElementsByTagName(), hoặc querySelector().

getElementById() Trả về phần tử có id cụ thể.

Sao chép mã
let header = document.getElementById("header");
console.log(header);
getElementsByClassName()

Trả về một danh sách các phần tử có cùng lớp (class).

Sao chép mã
let paragraphs = document.getElementsByClassName("text");
console.log(paragraphs);
querySelector() và querySelectorAll()

Trả về phần tử đầu tiên phù hợp với CSS selector hoặc tất cả các phần tử phù hợp.

Sao chép mã
let firstParagraph = document.querySelector("p");
let allParagraphs = document.querySelectorAll("p");

2.2. Thay Đổi Nội Dung và Thuộc Tính Link to heading

Khi bạn đã truy cập được phần tử DOM, bạn có thể thay đổi nội dung của nó, thay đổi các thuộc tính, hoặc thậm chí thay đổi các sự kiện.

Thay Đổi Nội Dung Sử dụng thuộc tính innerHTML hoặc textContent để thay đổi nội dung của phần tử.

Sao chép mã
let header = document.getElementById("header");
header.innerHTML = "Chào mừng đến với trang web mới!";

Thay Đổi Thuộc Tính Để thay đổi thuộc tính của một phần tử, bạn có thể sử dụng setAttribute().

Sao chép mã
let image = document.getElementById("image");
image.setAttribute("src", "new-image.jpg");

Hoặc bạn có thể thay đổi thuộc tính trực tiếp:

Sao chép mã
let button = document.getElementById("myButton");
button.disabled = true;

Thêm hoặc Xóa Lớp (Class) Để thêm hoặc xóa lớp cho phần tử, bạn có thể sử dụng classList.

Sao chép mã
let element = document.getElementById("element");
element.classList.add("new-class");  // Thêm lớp
element.classList.remove("old-class");  // Xóa lớp

2.3. Thêm và Xóa Phần Tử Link to heading

Bạn có thể tạo phần tử mới và thêm vào DOM, hoặc xóa phần tử khỏi DOM.

Thêm Phần Tử

Sao chép mã
let newParagraph = document.createElement("p");
newParagraph.textContent = "Đây là đoạn văn bản mới!";
document.body.appendChild(newParagraph);

Xóa Phần Tử

Sao chép mã
let paragraph = document.getElementById("removeParagraph");
paragraph.remove();

3. Xử Lý Sự Kiện DOM Link to heading

DOM cho phép bạn thêm các sự kiện vào các phần tử HTML. Một sự kiện có thể là một hành động của người dùng, như nhấn chuột, di chuyển chuột, hoặc nhấn phím.

3.1. Thêm Sự Kiện Link to heading

Để thêm sự kiện vào phần tử, bạn sử dụng phương thức addEventListener().

Sao chép mã
let button = document.getElementById("myButton");
button.addEventListener("click", function() {
    alert("Button clicked!");
});

3.2. Loại Bỏ Sự Kiện Link to heading

Bạn có thể loại bỏ sự kiện bằng phương thức removeEventListener().

Sao chép mã
let button = document.getElementById("myButton");
function handleClick() {
    alert("Button clicked!");
}
button.addEventListener("click", handleClick);

// Loại bỏ sự kiện
button.removeEventListener("click", handleClick);

4. Ví Dụ Ứng Dụng Link to heading

Dưới đây là một ví dụ đơn giản về cách sử dụng DOM để thay đổi nội dung và xử lý sự kiện:

Sao chép mã
<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <title>DOM Example</title>
    <script>
        function changeContent() {
            document.getElementById("myText").innerHTML = "Chào mừng bạn đến với JavaScript!";
        }
    </script>
</head>
<body>
    <h1 id="myText">Đây là tiêu đề ban đầu</h1>
    <button onclick="changeContent()">Thay đổi nội dung</button>
</body>
</html>

Khi người dùng nhấn nút “Thay đổi nội dung”, nội dung của phần tử sẽ thay đổi thành “Chào mừng bạn đến với JavaScript!”.
