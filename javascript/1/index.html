JavaScript Cơ Bản

JavaScript Cơ Bản Link to heading

JavaScript là một ngôn ngữ lập trình phổ biến được sử dụng chủ yếu trong phát triển web. Nó có thể được chạy trên trình duyệt và server (với Node.js). Bài viết này sẽ giới thiệu các khái niệm cơ bản trong JavaScript, bao gồm khai báo biến, kiểu dữ liệu, cấu trúc điều kiện, vòng lặp, và hàm.

1. Khai Báo Biến trong JavaScript Link to heading

Trong JavaScript, có ba cách để khai báo biến: var, let, và const. Mỗi cách có một số đặc điểm và phạm vi sử dụng khác nhau.

  • var: Được sử dụng trong các phiên bản JavaScript cũ. Nó có phạm vi toàn cục (global) hoặc phạm vi hàm (function).
  • let: Được sử dụng trong ES6 (ECMAScript 2015), có phạm vi khối (block scope).
  • const: Cũng được sử dụng trong ES6, giống như let, nhưng giá trị của const không thể thay đổi sau khi gán.

Ví dụ: Link to heading 

var x = 10;    // Sử dụng var
let y = 20;    // Sử dụng let
const z = 30;  // Sử dụng const

2. Kiểu Dữ Liệu trong JavaScript Link to heading

JavaScript hỗ trợ một số kiểu dữ liệu cơ bản:

Số: Các giá trị số (integer hoặc floating point). Chuỗi: Dữ liệu văn bản (string). Boolean: Dữ liệu đúng hoặc sai (true/false). Null: Một giá trị không xác định. Undefined: Một biến chưa được gán giá trị. Object: Dữ liệu kiểu đối tượng (Object). Array: Một dạng đặc biệt của đối tượng chứa các phần tử có chỉ số. Ví dụ:

Sao chép mã
let number = 10;        // Kiểu số
let name = "John";      // Kiểu chuỗi
let isActive = true;    // Kiểu boolean
let person = null;      // Kiểu null
let unknown;            // Kiểu undefined
let arr = [1, 2, 3];    // Kiểu array

3. Cấu Trúc Điều Kiện trong JavaScript Link to heading

JavaScript cung cấp các cấu trúc điều kiện cơ bản để kiểm tra và xử lý các điều kiện logic.

Câu lệnh if:

Sao chép mã
let age = 18;

if (age >= 18) {
    console.log("Bạn đủ tuổi trưởng thành");
} else {
    console.log("Bạn chưa đủ tuổi trưởng thành");
}

Câu lệnh switch:

Sao chép mã
let day = 3;

switch (day) {
    case 1:
        console.log("Chủ Nhật");
        break;
    case 2:
        console.log("Thứ Hai");
        break;
    case 3:
        console.log("Thứ Ba");
        break;
    default:
        console.log("Ngày không hợp lệ");
}

4. Vòng Lặp trong JavaScript Link to heading

Vòng lặp giúp thực thi mã nhiều lần mà không cần phải viết lại đoạn mã.

Vòng lặp for:

Sao chép mã
for (let i = 0; i < 5; i++) {
    console.log(i);  // In ra các số từ 0 đến 4
}

Vòng lặp while:

Sao chép mã
let i = 0;
while (i < 5) {
    console.log(i);  // In ra các số từ 0 đến 4
    i++;
}
Vòng lặp for...of (Dành cho mảng):
javascript
Sao chép mã
let arr = ["Apple", "Banana", "Orange"];

for (let fruit of arr) {
    console.log(fruit);  // In ra "Apple", "Banana", "Orange"
}

5. Hàm trong JavaScript Link to heading

Hàm là một khối mã có thể tái sử dụng, được định nghĩa bằng từ khóa function.

Định nghĩa hàm cơ bản:

Sao chép mã
function greet(name) {
    console.log("Hello, " + name);
}

greet("John");  // In ra "Hello, John"
Hàm với giá trị trả về:
javascript
Sao chép mã
function add(a, b) {
    return a + b;
}

let result = add(5, 10);
console.log(result);  // In ra 15
Hàm mũi tên (Arrow Function):
javascript
Sao chép mã
const multiply = (a, b) => a * b;

console.log(multiply(2, 3));  // In ra 6

6. Sự kiện trong JavaScript Link to heading

JavaScript hỗ trợ xử lý sự kiện trong các trang web, chẳng hạn như khi người dùng nhấp chuột vào một phần tử hoặc nhập liệu vào ô văn bản.

Ví dụ về sự kiện click:

Sao chép mã
<button id="myButton">Click Me</button>

<script>
document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});
</script>

7. DOM trong JavaScript Link to heading

DOM (Document Object Model) là một giao diện lập trình ứng dụng (API) cho phép JavaScript tương tác với các phần tử HTML trong một trang web.

Thêm, xóa, sửa phần tử:

Sao chép mã
// Thêm phần tử mới vào DOM
let newElement = document.createElement("p");
newElement.innerText = "Đây là một đoạn văn mới!";
document.body.appendChild(newElement);

// Xóa phần tử khỏi DOM
let elementToRemove = document.getElementById("myButton");
elementToRemove.remove();
