1. Giới thiệu về Java Link to heading

Java là một ngôn ngữ lập trình phổ biến, được phát triển bởi Sun Microsystems (nay thuộc Oracle). Java được thiết kế để dễ học, đa nền tảng và mạnh mẽ.

Một trong những đặc điểm nổi bật của Java là WORA (Write Once, Run Anywhere) - viết một lần và chạy được trên mọi nền tảng.

Ví dụ: Chương trình Java cơ bản Link to heading 

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Xin chào, thế giới!");
    }
}

Chương trình trên sẽ in ra màn hình dòng chữ “Xin chào, thế giới!” khi được chạy. Đây là một ví dụ đơn giản nhất của một chương trình Java, giúp bạn hiểu cách Java thực thi mã nguồn của mình.

2. Cú Pháp Java Link to heading

Java có một cú pháp khá dễ học và rõ ràng. Các chương trình Java bắt đầu với một lớp, trong đó có phương thức main(), nơi mà mọi chương trình Java đều bắt đầu. Cú pháp cơ bản của Java có thể được minh họa qua ví dụ sau:

Ví dụ:

Sao chép mã
public class MyFirstProgram {
    public static void main(String[] args) {
        System.out.println("Chào mừng bạn đến với Java!");
    }
}

Các thành phần chính trong cú pháp: Lớp (Class): Trong Java, tất cả đều là đối tượng, và lớp là mô hình của các đối tượng. Mỗi chương trình Java phải chứa ít nhất một lớp. Phương thức main: Là điểm bắt đầu khi chương trình được chạy. Câu lệnh System.out.println: Dùng để in thông tin ra màn hình.

3. Kiểu Dữ Liệu trong Java Link to heading

Java cung cấp một loạt các kiểu dữ liệu để lưu trữ các giá trị khác nhau trong các biến. Các kiểu dữ liệu cơ bản bao gồm:

Các kiểu dữ liệu cơ bản: int: Dùng để lưu trữ các giá trị số nguyên. double: Dùng để lưu trữ các giá trị số thực. boolean: Dùng để lưu trữ giá trị đúng (true) hoặc sai (false). char: Dùng để lưu trữ một ký tự đơn. String: Dùng để lưu trữ chuỗi ký tự. Ví dụ:

Sao chép mã
public class DataTypesExample {
    public static void main(String[] args) {
        int age = 25;
        double price = 99.99;
        boolean isJavaFun = true;
        char grade = 'A';
        String greeting = "Chào bạn!";
        
        System.out.println("Tuổi: " + age);
        System.out.println("Giá: " + price);
        System.out.println("Java thú vị không? " + isJavaFun);
        System.out.println("Điểm: " + grade);
        System.out.println(greeting);
    }
}

Trong ví dụ trên, bạn sẽ thấy cách sử dụng các kiểu dữ liệu khác nhau trong Java như int, double, boolean, char, và String.

4. Biến trong Java Link to heading

Biến là tên đại diện cho một giá trị trong bộ nhớ. Bạn có thể sử dụng biến để lưu trữ các giá trị và thao tác với chúng.

Cách khai báo biến:

Sao chép mã
<kiểu_dữ_liệu> <tên_biến> = <giá_trị>;
Ví dụ:
java
Sao chép mã
public class VariableExample {
    public static void main(String[] args) {
        int number = 10;  // Khai báo và gán giá trị cho biến 'number'
        String name = "John Doe";  // Khai báo và gán giá trị cho biến 'name'

Trong Java, trước khi sử dụng biến, bạn phải khai báo kiểu dữ liệu của biến đó.

5. Vòng Lặp trong Java Link to heading

Vòng lặp cho phép bạn lặp lại một đoạn mã nhiều lần. Java hỗ trợ nhiều loại vòng lặp như for, while, và do-while.

Ví dụ sử dụng vòng lặp for:

Sao chép mã
public class LoopExample {
    public static void main(String[] args) {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Lặp " + i);
        }
    }
}

Trong ví dụ trên, vòng lặp for sẽ in ra “Lặp 1”, “Lặp 2”, “Lặp 3”, “Lặp 4”, và “Lặp 5”.

Vòng lặp while:

Sao chép mã
public class WhileLoopExample {
    public static void main(String[] args) {
        int i = 1;
        while (i <= 5) {
            System.out.println("Lặp " + i);
            i++;
        }
    }
}

Vòng lặp do-while:

Sao chép mã
public class DoWhileLoopExample {
    public static void main(String[] args) {
        int i = 1;
        do {
            System.out.println("Lặp " + i);
            i++;
        } while (i <= 5);
    }
}
